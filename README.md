#  Análise de Cancelamento de Clientes

##  Sobre o Projeto

Projeto de análise de dados feito em Python para descobrir por que os clientes estão cancelando o serviço de uma empresa.

##  O que eu fiz

- Analisei uma base de dados com informações de clientes
- Descobri que 56% dos clientes cancelaram
- Identifiquei os principais motivos dos cancelamentos
- Propus soluções que reduzem o cancelamento para menos de 20%

##  Principais Descobertas

**Problema 1:** Todos os clientes com contrato mensal cancelaram  
**Solução:** Oferecer descontos para contratos anuais

**Problema 2:** Clientes que ligam mais de 4 vezes para o suporte cancelam  
**Solução:** Resolver os problemas na primeira ligação

**Problema 3:** Clientes com mais de 20 dias de atraso cancelam  
**Solução:** Criar equipe para resolver atrasos rapidamente
